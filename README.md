## ✨ About The Project

**CarConfig** is a stunning, high-performance web application designed to provide a seamless car rental experience. Built with the cutting-edge tech stack of React and Vite, styled elegantly with Tailwind CSS, this platform brings the showroom directly to your screen.

## 🚀 Features

- 🏎️ **Extensive Car Collection:** Browse through our wide range of vehicles, from electric to luxury.
- ⚡ **Lightning Fast:** Powered by Vite for instant server starts and rapid hot module replacements.
- 🎨 **Modern UI/UX:** Fully responsive and beautifully designed utilizing Tailwind CSS.
- 🔍 **Easy Navigation:** Smooth routing through Home, About, Services, Collection, and Contact pages.

## 🛠️ Installation & Setup

Want to run the project locally? Follow these steps!

1. **Clone the repository**
   ```bash
   git clone https://github.com/DishantSaini55/CarConfig.git
   ```

2. **Navigate to the directory**
   ```bash
   cd CarConfig
   ```

3. **Install the dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

## 📁 Project Structure

```bash
📦 src
 ┣ 📂 Car         # Car assets and logic
 ┣ 📂 components  # Reusable UI components (Navbar, Footer, Hero, CarCard)
 ┣ 📂 pages       # Main application pages (Home, About, Collection, etc.)
 ┣ 📜 App.jsx     # Root application component
 ┣ 📜 index.css   # Global Tailwind styles
 ┗ 📜 main.jsx    # React entry point
```
